# Security Policy

## Supported versions

| Version | Supported |
| ------- | --------- |
| latest `main` | yes |
| older tags | no |

## Reporting a vulnerability

**Do not open public issues for security problems.**

Email: [create a GitHub security contact or use your email — update before publishing]

Include:

- Description and impact
- Steps to reproduce
- Affected version / commit
- Suggested fix (optional)

We aim to acknowledge within **72 hours** and provide a remediation timeline within **7 days** for confirmed issues.

## Scope

In scope: this repository’s game client, build pipeline, and GitHub Actions workflows.

Out of scope: third-party hosting (GitHub Pages CDN), browser extensions, and modified local builds.

## Safe disclosure

We follow coordinated disclosure. Please allow time to patch before public disclosure.
